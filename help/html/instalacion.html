3. Instalación de la aplicación de escritorio

Para instalar la aplicación de peluquería canina en el equipo del cliente, sigue estos pasos:

  1. Ejecuta el instalador proporcionado. Este incluye todos los archivos necesarios para que la aplicación funcione correctamente.
  2. Durante la instalación, se crearán las carpetas, accesos directos y configuraciones internas necesarias.
  3. Una vez finalizada, abre la aplicación desde el acceso directo en el escritorio o desde el menú de inicio.

Nota: La base de datos ya está creada y configurada. No es necesario que el cliente la cree manualmente.

4. Instalación de MySQL

La aplicación requiere que MySQL esté instalado en el equipo para poder conectarse a la base de datos. El cliente debe realizar lo siguiente:

  1. Descargar MySQL desde el sitio oficial: https://dev.mysql.com/downloads/
  2. Instalar MySQL Server siguiendo el asistente de instalación.
  3. Durante la instalación, configurar el usuario (por ejemplo, root) y la contraseña que se utilizará para acceder a la base de datos.
  4. Verificar que el servicio de MySQL esté activo y funcionando correctamente.

⚠️ Importante: La aplicación está configurada para conectarse automáticamente a la base de datos local. Si el cliente modifica el usuario, contraseña o puerto de MySQL, deberá actualizar esos datos en el archivo de configuración de la aplicación.

✅ Requisitos mínimos

  • Sistema operativo: Windows 10 o superior
  • Java Runtime Environment (JRE) versión 8 o superior
  • MySQL Server instalado y en ejecución
