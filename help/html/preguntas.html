Preguntas Frecuentes

App de escritorio para la peluquería canina: gestión de clientes, mascotas, roles y MySQL.

Sobre la aplicación

¿Qué puedo hacer con la aplicación? Registrar clientes y sus mascotas, guardar historial y gestionar usuarios.

¿Es solo para la peluquería canina? Sí, está diseñada para la gestión interna de la empresa.

¿Necesito internet para usarla? Solo si la base de datos está en un servidor remoto.

Roles y privilegios

Roles disponibles: Administrador, Recepcionista y Empleado.

Administrador: acceso completo y gestión de usuarios.

Recepcionista: registrar y consultar clientes.

Empleado: solo consulta clientes.

Base de datos

Los datos se guardan en una base de datos MySQL.

El administrador puede hacer copias de seguridad.

Si se pierde la conexión, la aplicación mostrará un error.

Uso y soporte

Inicio de sesión con usuario y contraseña asignados.

Si olvidas tu contraseña, el administrador puede restablecerla.

Las actualizaciones deben instalarse manualmente.
