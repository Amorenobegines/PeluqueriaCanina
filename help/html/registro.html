Descripción

Esta ventana permite crear una nueva cuenta para acceder a la aplicación de peluquería canina. Si aún no estás registrado, completa el formulario con tus datos personales y credenciales de acceso.

⚠️ Recuerda: todos los campos deben estar completos. Si falta alguno, el sistema mostrará un error y no permitirá continuar.

Roles disponibles

  • Administrador: acceso completo, gestión de usuarios y privilegios.
  • Recepcionista: puede registrar y consultar clientes.
  • Trabajador: acceso limitado, principalmente consulta de clientes y mascotas.

El rol será asignado por el administrador al momento de tu registro.
